Acompanhe todas as novidades, melhorias e correções a cada versão do Teki.

v1.3.0 12 de março de 2026
  • Novo Monitor de conexão com probes HTTP, TCP e PostgreSQL
  • Novo Dashboard de monitoramento com scorecard, gráfico de latência e heatmap
  • Melhoria Fallback automático entre modelos de IA (Gemini → GPT)
  • Correção Token de autenticação agora é renovado automaticamente antes de expirar
v1.2.0 28 de fevereiro de 2026
  • Novo Integração com AbacatePay para pagamentos recorrentes
  • Novo Painel administrativo com logs de atividade
  • Melhoria Seletor de modelo de IA na interface do chat
  • Correção Captura de screenshots agora funciona em monitores de alta resolução
v1.1.0 10 de fevereiro de 2026
  • Novo App desktop para Windows e macOS
  • Novo Integração com Slack, WhatsApp, Discord e Telegram
  • Melhoria Tempo de resposta reduzido para menos de 3 segundos
v1.0.0 15 de janeiro de 2026
  • Novo Lançamento inicial do Teki
  • Novo Chat com IA para suporte técnico
  • Novo Sistema de autenticação com verificação por email
  • Novo API REST documentada para integrações
